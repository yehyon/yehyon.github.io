Sketch Shortcuts

스케치 키보드 단축키 정리

Insert (객체 삽입 단축키)

  • New Artboard (새 아트보드 만들기)

    A

  • Slice (슬라이스)

    S

  • Rectangle (사각형)

    R

  • Rounded Rectangle (둥근 모서리 사각형)

    U

  • Oval (타원형)

    O

  • Vector Point (벡터 툴)

    V

  • Pencil (연필)

    P

  • Text (텍스트)

    T

Type (글씨 관련 단축키)

  • Bold (두껍게)

    Command + B

  • Italic (기울임)

    Command + I

  • Underline (밑줄)

    Command + U

  • Increase Font Size (글자 크기 키우기)

    Alt + Command (+) +

  • Decrease Font Size (글자 크기 줄이기)

    Alt + Command (+) -

  • Incerase Character Spacing (글자 간격 키우기)

    Alt + Control + L

  • Decrease Character Spacing (글자 간격 줄이기)

    Alt + Control + T

  • Change Font (글자꼴 바꾸기)

    Command + T

  • Convert Text to Outlines (텍스트 아웃라인으로 바꾸기)

    Shift + Command + O

  • Align Left (왼쪽 정렬)

    Command + Shift + {

  • Alight Center (가운데 정렬)

    Command + Shift + |

  • Alight Right (오른쪽 정렬)

    Command + Shift + }

  • Special Characters (특수 문자)

    Control + Command + Space

Canvas View (캔버스 뷰 단축키)

  • Zoom In (확대)

    Command (+) +

  • Zoom Out (확대에서 빠져나오기)

    Command (+) -

  • Actual Size (실제 크기)

    Command + 0

  • Center Canvas (켄버스 중앙으로 조정)

    Command + 1

  • Zoom Selection (선택 요소 확대)

    Command + 2

  • Center Selection (선택 요소를 캔버스 중앙으로)

    Command + 3

  • Temporary Zoom to Actual Size (임시 실제 사이즈로 확대)

    §

  • Focus on First Input Field (첫 인풋 필드로 포커스)

    Alt + Tab

  • Toggle Rulers (자 켜고 끄기)

    Control + R

  • Toggle Grid (그리드 켜고 끄기)

    Control + G

  • Toggle Layer Guides (레이어 켜고 끄기)

    Control + L
