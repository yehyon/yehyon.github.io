Poptip Zipline

Unlike Twitter Ziplines, Instagram Ziplines tracked two different kinds of data: hashtags and images. When the user creates an Instagram Zipline to track a certain hashtag, Poptiip Zipline would aggregate images that contain the target hashtag and also other hashtags used with the target hashtag. Below are two Instagram Ziplines that are tracking mascara and MAC cosmetics respectively.

Mascara Zipline Live
MAC Cosmetics Zipline Live

When a user clicks on the Zipline, one goes into the detail view. The main goal of the detail view was to allow users to see as much data possible in a single view while keeping the inflow of data as less visually jarring as possible.
