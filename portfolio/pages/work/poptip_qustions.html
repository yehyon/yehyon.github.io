Poptip Questions

A Realtime Polling App That Aggregates Crowd Voice Through Twitter & Facebook

Devlopment of Questions

Poptip Questions was a social media polling app that could be used with either Facebook or Twitter. The dashboard allowed users to post a new questions or visit their past questions.

Questions Dashbaord Empty State

The biggest challenge was creating an experience where the user can learn while using the product for the first time. We tried to accomplish this by using
1. clear call to action texts
2. using action colors for affirmation
3. affirmative dynamic changes such as when a user adds answers in their question the poptip dashboard would automatically add those answers into the below answers form.
